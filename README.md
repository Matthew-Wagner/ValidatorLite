<h1>Welcome to ValidatorLite</h1>
<p><em>ValidatorLite</em> is a light weight, single file HTML 5 / javascript library for adding validation to HTML 5 forms.</p>
<p>While <a href="https://jquery.com/">jQuery</a> and <a href="https://getbootstrap.com/">Bootstrap</a> are not required, this sample file does make use of them.</p>
<p>
    The library itself uses the HTML 5 <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Guides/Constraint_validation" target="_blank">Constraint Validation API</a>
    but expands on it with some new attributes.
</p>
<p>For further documentation and examples, click <a href="ValidatorLiteReadMe.html">here</a></p>
